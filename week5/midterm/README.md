# Student Tracker Midterm
**Name:** Tuesdae Hunter
**Class:** C++

## What it does
A program that manages student records. You can add students, 
view their grades and average, and search for students by name.

## How to run it
g++ main.cpp -o main
./main

## Features
1. **Add Student** - Enter a student name, ID, and up to 5 grades
2. **View All Students** - Displays all students and their calculated grade average
3. **Search Student** - Search for a student by name
4. **Quit** - Exits the program1
